Step 2 of 4

Install reminder

If you opened this from a DMG, drag Hotate into Applications first, then launch from there. Eject the disk image when you’re done.

First open? macOS may warn about an unidentified developer—use Right-click → Open once, or allow it under Privacy & Security.

Step 3 of 4

How to use it

  • Click the shell icon in the menu bar to open this window.
  • Use Settings to pick the project folder Hotate should reason about.
  • Type full sentences: “What’s in this folder?” or “Move these into Travel.”
  • Read-only replies appear as chat; risky actions show a command to review first.
